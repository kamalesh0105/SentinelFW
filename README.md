**SentinelFW** is a lightweight Linux firewall automation and intrusion detection tool built on **UFW**.  
It focuses on securing hosts by automating firewall rules and detecting SSH brute-force attacks through log analysis.

---

## Features
- UFW firewall automation  
- Blocking insecure services  
- SSH rate limiting  
- Log-based brute-force detection  
- Firewall logging support  

---

## Tech Stack
- Bash  
- Linux (Ubuntu/Debian)  
- UFW / Netfilter  

---

## Learning Focus
- Linux firewall fundamentals  
- Host-based security hardening  
- Blue Team / SOC workflows  
- Secure Bash scripting
